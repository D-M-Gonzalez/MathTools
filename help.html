Math Tools

Una herramienta simple para realizar tus cálculos

¿Cómo usar esta herramienta?

MathTools es una calculadora especialmente diseñada para devolvernos el paso a paso de la resolución.

Para utilizarla uno escribe directamente en pantalla lo que quiera calcular, pero respetando ciertas reglas:

  • Las operaciones se escriben con los símbolos + para la suma, - para la resta, * para la multiplicación, / para la división y ^ para las potencias/raices
  • Todas las operaciones se deben escribir, no puede haber operaciones implícitas como sucede a veces que uno no pone la multiplicación previo a un paréntesis
  • La separación en términos se realiza automáticamente, aún así, es posible separar utilizando paréntesis, corchetes y llaves
  • Los paréntesis (), corchetes [] y llaves {} deben ir de a pares, en caso de que no se cierren o alguno no este de a pares la calculadora devolverá un error
  • Siempre debe haber una operación antes y despues de un término separado por los mismos de arriba
  • Una vez terminado, se presiona el símbolo = marcado en celeste para realizar el cálculo
  • En caso de no encontrar un símbolo o no tenerlo en el teclado, hay un teclado manual debajo de la zona de escritura donde se puede ingresar manualmente

Además de los calculos, en la sección izquierda se pueden observar ejemplos que se pueden utilizar copiando o presionando el símbolo = en cada uno de los ejemplos

Del lado derecho de la zona de cálculo tenemos un historial donde se graban cada uno de los cálculos válidos que realizamos, esta misma se puede volver a utilizar presionando el símbolo = o podemos borrar alguno presionando la X.

También se puede eliminar todo el historial presionando en limpiar.
